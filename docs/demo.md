## Five-minute flow

Frame the fragmented land-and-labour problem, browse verified land, open AI analysis and crop planning, show weather/disease intelligence, hire a worker, send a chat message, then approve a listing in admin.

## Seven-minute flow

Add farmer application and negotiation, agreement generation, farm task planning, notification delivery, and the role-specific dashboards.

## Ten-minute flow

Add remote monitoring, activity history, moderation audit trail, failure fallback explanation, architecture/security overview, business model, and roadmap.

Seed with `npm run seed --workspace backend`. Existing local/demo providers should be selected before judging. Test the exact route order once on the deployment and keep a local seeded fallback available.
